# peegee

> Postgres + json queries + transaction support with promises


## Installation

- Requires es6.

**1. Create .env file**  
- Setup database connection variables.  
- I recommend heroku free tier.  

**2. Create and seed database**  
`migrate up`

**3. Run test**  
`npm test`
